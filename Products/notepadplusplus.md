# Preferred Theme
- Dracula
  - https://draculatheme.com/notepad-plus-plus
  - C:\Users\YourUsername\AppData\Roaming\Notepad++\themes
  - Copy the downloaded theme file (Dracula.xml or Alucard.xml) into this themes folder.
  - Restart Notepad++
  - Go to Settings > Style Configurator > Theme and select your theme from the dropdown menu.
  - Change Global Styles > Active Tab text to black
  

If you can't install an addon...
- Settings > Style Configurator
- Select theme: Solarized
- Check "Enable global background colour"
- Set background to Black
- Save and Close

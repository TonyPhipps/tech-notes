# Make Text 3d
- Enter Object Mode
- Shift+A > Text
- TAB > Type in the text you want
- Navigate in bottom right to Data (letter a)
- Transform > Set Size
- Font > Select Font
- Geometry > set Extrusion
- Export as .STL
